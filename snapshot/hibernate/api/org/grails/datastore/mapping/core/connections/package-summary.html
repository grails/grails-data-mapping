  • Interface Summary 
    Interface Description
    ConnectionSource Represents a connection source, which could be a SQL DataSource, a MongoClient etc.
    ConnectionSourceFactory A factory for creating new ConnectionSource instances
    ConnectionSources Models multiple connection sources
    ConnectionSourcesListener A listener for connection sources
    ConnectionSourcesProvider An interfaces for classes that provide ConnectionSources
    MultipleConnectionSourceCapableDatastore A Datastore capable of configuring multiple Datastore with individually named ConnectionSource instances
