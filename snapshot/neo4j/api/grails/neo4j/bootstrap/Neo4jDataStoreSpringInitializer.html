[Groovy] Class Neo4jDataStoreSpringInitializer

      • grails.neo4j.bootstrap.Neo4jDataStoreSpringInitializer

      • An AbstractDatastoreInitializer used for initializing GORM for Neo4j when using Spring. If you are not using the Spring container then using the constructors of org.grails.datastore.gorm.neo4j.Neo4jDatastore is preferable
