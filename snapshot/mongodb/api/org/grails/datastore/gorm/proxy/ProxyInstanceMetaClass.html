Package: org.grails.datastore.gorm.proxy

[Java] Class ProxyInstanceMetaClass

      • org.grails.datastore.gorm.proxy.ProxyInstanceMetaClass 

      • public class ProxyInstanceMetaClass
extends groovy.lang.DelegatingMetaClass

        Per-instance metaclass to use for proxied GORM domain objects. It auto-retrieves the associated entity when fields, properties or methods are called (other than those supported by the proxy). The methods and properties supported by the proxy are:

        • id/getId() - no resolve performed
        • initialized/isInitialized() - no resolve performed
        • class/getClass() - no resolve performed
        • metaClass/getMetaClass() - no resolve performed
        • domainClass/getDomainClass() - no resolve performed
        • target/getTarget() - resolve performed
        • initialize() - resolve performed
        Authors:
        Tom Widmer
