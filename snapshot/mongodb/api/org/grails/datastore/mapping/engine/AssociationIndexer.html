      public interface AssociationIndexer<K, T>
extends AssociationQueryExecutor

      Responsible for creating indices for associations used in queries. An instance may be specific to a particular association of a particular native instance of an entity (the parent of the association).

        • public void preIndex(K primaryKey, java.util.List<T> foreignKeys)

          Creates an index queryable via the primary key. This is called *before* the entity that this association indexer is part of is persisted, but after the native entry has been updated ready to be persisted. This allows the index to be placed in the native instance itself, e.g. in a document database. Usually, for a particular association type, only this OR index(Object, java.util.List) will be implemented.

          Parameters:
          primaryKey - The primary key
          foreignKeys - The foreign keys
