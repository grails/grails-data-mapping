(Quick Reference)

redis

Purpose

A Spring bean that provides access to the lower level Redis API

Examples

def redis
def foo = {
    redis.select(2)
    redis.flushdb()
    redis["message"] = "World"
    render "Hello ${redis["message"]}!"
}

Description

See the org.grails.redis.util.RedisTemplate interface for the full list of available methods.
