      • abstract rx.Observable<D> first(java.util.Map queryParams)

        Finds the first object. If queryParams includes 'sort', that will dictate the sort order, otherwise natural sort order will be used. queryParams may include any of the same parameters that might be passed to the list(Map) method. This method will ignore 'order' and 'max' as those are always 'asc' and 1, respectively.

        Returns:
        the first object in the datastore, null if none exist
