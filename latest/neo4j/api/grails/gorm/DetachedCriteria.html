[Groovy] Class DetachedCriteria<T>

      • grails.gorm.DetachedCriteria
      • All Implemented Interfaces and Traits:
        GormOperations<T>, java.lang.Iterable<T>, QueryableCriteria<T>

        @groovy.transform.CompileStatic
class DetachedCriteria<T>
extends AbstractDetachedCriteria<T>
implements GormOperations<T>, java.lang.Iterable<T>, QueryableCriteria<T>

        Represents criteria that is not bound to the current connection and can be built up and re-used at a later date.

        Authors:
        Graeme Rocher
        Since:
        1.0
